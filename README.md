Practical Networking: Documentation and simulations of CCNA-level labs exploring OSPF, VLANs, ACLs, and more.
# My Networking Lab Journey 🚀

Welcome to my networking repository! This project serves as a digital portfolio of my hands-on experience while studying for the **Cisco Certified Network Associate (CCNA)**. 

## 📂 Repository Structure
- **01-Basic-Config**: Initial setup, passwords, SSH, and banner configurations.
- **02-Switching**: VLANs, Trunking, VTP, and Spanning Tree Protocol (STP).
- **03-Routing**: Static routing, OSPF, and Inter-VLAN routing.
- **04-Security**: Port Security, Access Control Lists (ACLs), and NAT.

## 🛠️ Tools Used
- **Cisco Packet Tracer**
- **Wireshark** (for packet analysis)
- **Linux Terminal**

## 🎯 Goals
The goal is to master network infrastructure design, troubleshooting, and security implementation.
